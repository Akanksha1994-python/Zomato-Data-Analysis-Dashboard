# Zomato-Data-Analysis-Dashboard
This project is a Power BI dashboard based on Zomato data, aimed at analyzing restaurant trends, customer preferences, and key business insights.

📊 Features
Restaurant distribution by location
Cuisine analysis
Ratings and price range insights
Online delivery & table booking trends
Interactive dashboard visuals
🛠️ Tools Used
Power BI
Data Cleaning & Transformation
Data Visualization
